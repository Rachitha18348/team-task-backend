Project Name: Team Task Manager

Tech Stack:
- React.js
- Node.js
- Express.js
- MongoDB

Features:
- User Signup/Login
- JWT Authentication
- Create Tasks
- Update Tasks
- Delete Tasks

Frontend:
- React + Vite + Tailwind CSS

Backend:
- Express.js REST APIs

Database:
- MongoDB Atlas

How to Run Locally:

Backend:
cd backend
node server.js

Frontend:
cd frontend
npm run dev
